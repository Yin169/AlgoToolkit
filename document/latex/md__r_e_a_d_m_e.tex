Welcome to the LAToolkit, a C++ library designed to provide a robust and efficient set of tools for performing linear algebra operations. This repository contains two core header files\+: {\ttfamily Matrix\+Obj.\+hpp} and {\ttfamily basic.\+hpp}, which together form the foundation of our linear algebra capabilities.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{md__r_e_a_d_m_e_autotoc_md1}
The library consists of several components\+:\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysubsection{\texorpdfstring{Matrix\+Obj.\+hpp}{Matrix\+Obj.\+hpp}}\label{md__r_e_a_d_m_e_autotoc_md2}
This file defines the {\ttfamily Matrix\+Obj} and {\ttfamily Vector\+Obj} classes, which are templates that allow for the creation and manipulation of matrices and vectors with arbitrary data types. The {\ttfamily Matrix\+Obj} class includes methods for matrix addition, subtraction, scalar multiplication, and matrix multiplication, as well as transpose operations. The {\ttfamily Vector\+Obj} class, which inherits from {\ttfamily Matrix\+Obj}, extends these capabilities to one-\/dimensional arrays, providing additional methods for vector normalization and dot product calculations.\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysubsection{\texorpdfstring{Basic.\+hpp}{Basic.\+hpp}}\label{md__r_e_a_d_m_e_autotoc_md3}
The {\ttfamily basic.\+hpp} file contains a namespace {\ttfamily basic} that encompasses a variety of linear algebra algorithms and utility functions. These include\+:


\begin{DoxyItemize}
\item {\bfseries{Power Iteration}}\+: An iterative method for finding the dominant eigenvalue and its corresponding eigenvector of a matrix.
\item {\bfseries{Rayleigh Quotient}}\+: A method for estimating an eigenvalue of a matrix given an eigenvector.
\item {\bfseries{Subtractive Projection}}\+: A technique for orthogonalizing vectors.
\item {\bfseries{Gram-\/\+Schmidt Process}}\+: An algorithm for orthonormalizing a set of vectors.
\item {\bfseries{Householder Reflections}}\+: A method for constructing orthogonal matrices, used in the QR decomposition.
\item {\bfseries{QR Decomposition}}\+: A factorization of a matrix into an orthogonal matrix Q and an upper triangular matrix R.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md4}{}\doxysubsection{\texorpdfstring{Iter\+Solver.\+hpp}{Iter\+Solver.\+hpp}}\label{md__r_e_a_d_m_e_autotoc_md4}
Contains template classes for iterative solvers such as Gradient Descent and Jacobi methods, which inherit from the Iter\+Solver\+Base class. \hypertarget{md__r_e_a_d_m_e_autotoc_md5}{}\doxysubsection{\texorpdfstring{Solver\+Base.\+hpp}{Solver\+Base.\+hpp}}\label{md__r_e_a_d_m_e_autotoc_md5}
A template base class for iterative solvers, defining common properties and a pure virtual function for gradient calculation.\hypertarget{md__r_e_a_d_m_e_autotoc_md6}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md6}

\begin{DoxyItemize}
\item {\bfseries{Template-\/\+Based}}\+: Works with any data type that supports the necessary operations.
\item {\bfseries{Efficiency}}\+: Optimized for performance with in-\/place operations where possible.
\item {\bfseries{Extensibility}}\+: Easily add new algorithms and functions to the library.
\item {\bfseries{Iterative Solvers}}\+:: Implements common iterative methods for solving linear systems.
\item {\bfseries{Preconditioning}}\+:: Supports preconditioning in iterative solvers to improve convergence.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md7}{}\doxysection{\texorpdfstring{Getting Started}{Getting Started}}\label{md__r_e_a_d_m_e_autotoc_md7}
To get started with LAToolkit, simply clone this repository and include the {\ttfamily Matrix\+Obj.\+hpp} and {\ttfamily basic.\+hpp} files in your project.
